Issue catalogue

Issue 1 — Q15/Q16 ČJA geographic filtering: locality codes span all Bohemia/Moravia (MEDIUM)

Appears in: Q15, Q16 Occurrences: 2 (persistent across all evals)

The text_enriched column substitutes all 477 locality codes inline. The model receives the full variant list including Slezsko forms (Spálov/Nový Jičín, Morávka/Frýdek-Místek) when asked specifically about Brněnsko. It cites these as Brněnsko examples — a geographic error. The model cannot filter localities without a [SQLQUERY:] tool.

Q16 answer claims: "z Brněnska konkrétně jsou doloženy: ďevče (Spálov/Nový Jičín)"
Actual: Spálov is ~200 km from Brno

Root cause: structural ČJA limitation. Mitigation: a [SQLQUERY:] tool against body_site.okres would allow WHERE okres IN ('Brno-venkov','Brno-město','Blansko','Vyškov').

Issue 2 — Q09/Q11 multi-word phrase saturation (MEDIUM)

Appears in: Q9 (21/24 = 88% multi-word), Q11 (7/12 = 58%), Q06 (11/22 = 50%), Q14 (12/22 = 55%), Q25 (7/11 = 64%) Wasted slots: approx. 30 across the batch

Multi-word FTS queries are preprocessed by _preprocess_fts_query() and often return useful results (ASSC FTS handles them). However, many ftsearch calls still use natural-language phrases ("krojové odívání a jejich součásti") that get tokenized into individual unrelated terms. The budget is consumed exploring broad FTS without targeted follow-ups.

Issue 3 — Q09 SNČJ phraseme discovery gap (MEDIUM)

Appears in: Q9 Occurrences: persistent across evals

Q9 searches ASSC (21/24 lookups) and barely touches SNČJ (3 lookups, only chlast* fruitfully). SNČJ has phraseme sections in its definition JSON for entries like pít, opilý, chlastat. The structured rendering would surface these if the model searched SNČJ directly. The model is not prompted to check SNČJ phraseme sections for alcohol/drinking vocabulary.

Issue 4 — Q11 germanismus FTS failure (MEDIUM)

The correct strategy for Q11 is ftsearch "z něm" | sncj → enumerate entries with German etymology markers. The vsearch fallback found 3 plausible germanisms via semantic search. Full coverage of SNČJ A–D germanisms requires knowing to search the etymology markers explicitly.
